本地es6 转 es5

  1. npm init 生成package.json
  2. Babel-cli => npm install -g babel-cli
  3. npm install --save-dev babel-preset-es2015 babel-cli
  4. 新建.babelrc文件
  5. 执行命令：babel src/index.js -o dist/index.js
  6. 简化：pagejson文件"scripts": { "build": "babel src/index.js -o dist/index.js" },
  7. npm run build
