1. 全局安装webpack: npm install webpack -g
  2. cd 到文件目录初始化：npm init; npm install --save-dev wenpack 安装
  3. 通过webpack-dev-server实现页面的自动刷新.安装：npm install --save-dev webpack-dev-server
  4. 安装loader，stylus以及postCss

    npm install style-loader css-loader stylus-loader stylus --save-dev

    npm install --save-dev postcss-loader autoprefixer （自动添加css 前缀的插件 -webkit -o -moz）

  5. 安装path备用： npm install --save-dev path
  6. 时时改变
